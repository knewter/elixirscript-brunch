# ElixirScript Brunch Plugin

This plugin for brunch adds support for ElixirScript files.

It will look for files with the extensions `ex`, `exs`, or `exjs` in your watched paths

It depends on the `ex2js` binary being in your path.  You can install it from
here: [elixirscript](https://github.com/bryanjos/elixirscript)

Usage with Phoenix:

* Grab elixir.js from the latest elixirscript release [here](https://github.com/bryanjos/elixirscript/releases) and place it in `web/static/vendor`
* Install elixirscript-brunch plugin
  ```bash
  npm install --save elixirscript-brunch
  ```
